TRAINING
1. Download and install Miniconda (From Miniconda site or this URL)
        https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
2. Clone Wenet main repo and install dependencies (pip install -r requirements.txt && pip install -e .)
        git clone https://github.com/wenet-e2e/wenet.git
3. Within Wenet repo, use the option to download, split and prepare the AISHELL dataset.
4. Run training (Change num_workers & datapath in run.sh as necessary)
        https://github.com/wenet-e2e/wenet/tree/main/examples/aishell/s0

INFERENCE
1. Download and install Miniconda (From Miniconda site or this URL)
        https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

2. Clone sherpa_onnx repo modeled on wenetspeech
        git clone https://huggingface.co/csukuangfj/sherpa-onnx-zh-wenet-wenetspeech
        << This is a CTC model therefore only capable of 1-best output >>

3. pip install -U sherpa-onnx soundfile numpy

4. Use asr_offline_wenet.py to generate the initial ASR prediction of the speech file (Edit path for speech file, asr model and tokenizer as required)
        On GPU inference --> Use `model.onnx`
        On CPU inference --> Use `model.int8.onnx`

5. Get API key from a chinese LLM provider. Used the below for this task.
        pip install zhipuai
        LLM --> https://open.bigmodel.cn/
        API --> https://bigmodel.cn/usercenter/proj-mgmt/apikeys --> 8446ac418fc747d08ebe1e60e24d0733.YoOoPTf6ftBepD4b

6. Edit fix_asr_text.py with the API key and relevant details and run: python asr_offline_wenet.py
